# ✅ Repository Ready - Push from Cursor

I've set up everything! The repository is connected. Now push using Cursor's built-in Git:

## Push from Cursor (Handles Authentication Automatically)

1. **In Cursor:** Press `Cmd+Shift+G` to open Source Control
2. **You should see:** "main" branch with an up arrow (↑) or "Publish Branch" button
3. **Click:** The up arrow (↑) or "Publish Branch" button
4. **Cursor will:** Handle GitHub authentication automatically
5. **Done!** Code will be pushed

OR

## Use Cursor's Command Palette

1. Press `Cmd+Shift+P`
2. Type: `Git: Push`
3. Press Enter
4. Cursor will handle authentication

---

## Then Deploy to Vercel:

1. **Go to:** https://vercel.com/new
2. **Click:** "Import Git Repository"
3. **Select:** `troyknott/aicv-troy-knott-profile`
4. **Click:** "Deploy"
5. **Add Environment Variable:**
   - Key: `NEXT_PUBLIC_SITE_URL`
   - Value: Your Vercel URL (shown after deploy)
6. **Redeploy** after adding the variable
