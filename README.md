# Troy Knott - Personal Profile Site

AI-optimized personal profile site built with Next.js 15.5, optimized for AI crawlers and LLM citations.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## AI Optimization Features

- Server-side rendering (SSR) for all content
- Comprehensive JSON-LD structured data (Person, FAQPage schemas)
- Strategic robots.txt configuration for AI crawlers
- Semantic HTML5 structure
- Fast page load times
- Mobile responsive design
